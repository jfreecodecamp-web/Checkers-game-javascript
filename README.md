
# Checkers Game (JavaScript)

## Description
A browser-based Checkers game developed using JavaScript. The game includes core mechanics such as piece movement, capturing, and win condition detection.

## Features
- Piece movement logic
- Capture mechanics
- Turn-based gameplay
- Win condition detection

## Tech Used
- JavaScript
- HTML
- CSS

## Live Demo
https://jfreecodecamp-web.github.io/ludo-checkers-javascript/

## How to Run
1. Download or clone the repository
2. Open index.html in your browser
